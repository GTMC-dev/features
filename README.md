# Features Catalogue

Backend for features on GTMC website. All data under issues in this repository.

## *"Deleting"* an Feature

Simply close the corresponding issue. The feature will be hidden on the website.

## Opening Features Manually

It is planned to allow creating features directly through creating an issue under this repo.

Before that, we must ensure that would be compatible with the current local indexing (parallel from GitHub's indexing). Also, a suitable issue template should be created.
